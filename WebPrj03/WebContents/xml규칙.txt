XML 규칙
1. 대소문자 구분
2. 모든 TAG 는 쌍(pair)으로 작업
   <h1></h1>  
   <br />  : ok
     <br>    : not ok   
3. 모든 속성은 name="value" 로 표기한다 
   <input type="radio" checked >            : not ok
   <input type="radio" checked="checked" />  : ok
4. xml 반드시 .. 로 시작한다
<?xml version="1.0" encoding="utf-8" ?>
5. 한개의 문서(document) 는 한개의 root element만 존재한다 
6. 주석(<!-- -->)과 xml 선언은 태그안에 위치할 수 없다
